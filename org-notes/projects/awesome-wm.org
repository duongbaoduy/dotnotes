* Shortkeys
  <W> = <super> or <window>
  <S> = <Shift>
  <C> = <Ctrl>
  <M> = <Meta> or <Alt>
  | Key               | Function | Describe                            |
  |-------------------+----------+-------------------------------------|
  | <W> + <S> + <num> |          | Move current window to screen <num> |
  | <W> + <S> + <r>   |          | Restart AwesomeWM                   |
  | <W> + <C> + <q>   |          | Quit AwesomeWM                      |
